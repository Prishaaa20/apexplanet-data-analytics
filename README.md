# ApexPlanet Data Analytics Internship - Task 1

## Project Overview
This project focuses on Data Cleaning and Exploratory Data Analysis (EDA) using the Sample Superstore dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code

## Tasks Performed
- Loaded dataset into Pandas DataFrame
- Checked and handled missing values
- Removed duplicate records
- Converted categorical columns to category data type
- Handled outliers using IQR method
- Saved cleaned dataset
- Performed Exploratory Data Analysis (EDA)
- Created Histogram, Boxplot, Bar Chart, Heatmap and Line Chart

## Files
- notebooks/task1_EDA.ipynb
- data/raw/SampleSuperstore.csv
- data/processed/cleaned_SampleSuperstore.csv

## Key Insights
- No missing values were found.
- 17 duplicate records were removed.
- Office Supplies is the most frequently purchased category.
- Sales and Profit have a positive correlation.
- Higher discounts tend to reduce profit.
